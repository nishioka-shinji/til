## 01.監視アーキテクチャの概要
### なぜ監視が必要なのか？
システムのリソース不足、アプリケーションの稼働を確認することで、ユーザー体験を損なわないようにするため
### チェックすべき代表的な監視項目
1. リソース監視
2. ログ監視
3. APM
4. 外形監視
5. セキュリティ
6. コスト
